# PID Controller


피드백 제어기의 대표적 제어 기법

지령대비 출력 오차(e(t) = r(t)-y(t))를 처리하는 방식에 따른 제어기

시간 응답을 확인하며 제어기 Tuning 가능, 구현 용이, 직관적, Trial& Error 필요, 숙련도 필요
